# Spatial Data Analytics

Learning repository for Spatial Data Analytics, Location Intelligence, Mobility Analysis, and Retail Geography using Python.

---

## Why This Repository Exists

Most business decisions happen somewhere.

Stores have locations.

Customers travel through cities.

Malls compete for catchment areas.

Coffee shops compete for attention within walking distance.

This repository explores how location influences business outcomes using Python-based geospatial analysis.

The goal is not to become a GIS specialist.

The goal is to answer business questions using spatial data.

---

## Learning Roadmap

### Phase 1 — Geospatial Foundations

Objective:

Understand how spatial data works.

Topics:

* Coordinates
* CRS (Coordinate Reference Systems)
* Point, Line, Polygon
* GeoDataFrames
* Spatial Joins
* Buffer Analysis

Tools:

* GeoPandas
* Shapely

Projects:

* Load Indonesia administrative boundaries
* Plot Jakarta districts
* Count POIs per district
